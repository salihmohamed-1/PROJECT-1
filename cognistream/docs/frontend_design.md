# CogniStream Frontend Design

## Overview

The CogniStream frontend provides a clean dashboard for monitoring developer productivity. It is built using React and Vite to ensure a fast and responsive user interface.

---

## Dashboard Layout

The dashboard contains the following sections:

- Header
- Productivity Cards
- Flow State Indicator
- GitHub Activity
- Slack Activity
- IDE Activity

---

## Technology Stack

- React
- Vite
- JavaScript
- CSS

---

## Components

### Header

Displays the project title.

### Dashboard Cards

Shows:

- GitHub Commits
- Slack Messages
- Focus Hours
- Flow Score

### Future Enhancements

- Dark Mode
- Charts
- Real-time API integration
- User Authentication

---

## Expected Workflow

Mock APIs → Processing Pipeline → FastAPI → React Dashboard

The dashboard consumes processed developer metrics and displays them in a user-friendly format.
